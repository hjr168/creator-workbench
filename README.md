# Personal IP Content Workbench

A local-first interactive content production workbench for planning topics, confirming outlines, calling generation tools, and reviewing publication results.

## Current Phase

Phase 2: Next.js, TypeScript, and Tailwind CSS app bootstrap with a minimal runnable dashboard.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Core Principle

Content generation must follow confirmation:

```text
idea -> topic -> angle -> title -> outline -> confirmed input -> tool generation
```
